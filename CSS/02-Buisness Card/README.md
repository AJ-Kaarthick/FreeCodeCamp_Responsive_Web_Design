# 🎴 Project 02: Business Card

An independent responsive CSS layout showcasing card UI structuring, typography hierarchy, image constraints, and clean hyperlinking patterns.

## 🛠️ Key Architectural Constraints Mastered

1. **Card Container Box Model:**
   * Standardized container dimensions with explicit width (`300px`), centered auto-margins (`margin-left: auto; margin-right: auto`), and top offset spacing (`margin-top: 100px`).

2. **Typography & Global Inheritance:**
   * Defined base document font stacks (`font-family: Arial, sans-serif`) and background parameters (`background-color: rosybrown`) at the `body` level for clean property inheritance.

3. **Responsive Image & Anchor Normalization:**
   * Constrained image dimensions using relative scaling bounds (`max-width: 100%`) and normalized link aesthetics using `text-decoration: none`.
