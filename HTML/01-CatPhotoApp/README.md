# 🐱 Workshop 01: Cat Photo App

This is the very first interactive workshop in the freeCodeCamp Responsive Web Design curriculum. It focuses on building a simple layout to become familiar with basic structural elements and coding syntax.

## 📝 Project Description
The Cat Photo App introduces the structure of a standard webpage. It demonstrates how to utilize tags to organize regular content like paragraphs, emphasize specific text, display clean images with custom descriptions, build navigations via anchor tags, and construct organized lists.

## 🧠 Core Technical Takeaways
* **Semantic HTML Structure:** Practiced building an clear hierarchy using layout blocks like `<main>`, `<section>`, and `<footer>`.
* **Media Handling:** Implemented self-closing tags like `<img>` alongside proper descriptive text (`alt` attribute) for screen readers and accessibility.
* **Content Organization:** Mastered building ordered (`<ol>`) and unordered (`<ul>`) lists, as well as structural text groupings (`<figure>` and `<figcaption>`).
* **Hyperlinks:** Learned how to transform text and standalone images into functional hyperlinks using both internal relative paths and external absolute URLs.
